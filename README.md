# Pawsome Shelter – Animal Adoption Website

Welcome to **Pawsome Shelter**, an animal adoption platform designed to connect rescued animals with loving homes. This project was built using **Jekyll** and hosted with **GitHub Pages**.



##  Live Website

 https://smushy14.github.io/



##  Project Overview

Pawsome Shelter aims to:

* Help animals find safe and loving homes
* Promote responsible pet adoption
* Connect volunteers and adopters with shelters

---

## Technologies Used

* **Jekyll** – Static site generator
* **GitHub Pages** – Hosting platform
* **Markdown & HTML** – Content and structure
* **Architect Theme** – Site design

---

##  Website Pages

###  Home

* Landing page introducing the platform

###  Animals

* Displays available animals for adoption

###  Adopt

* Lists adoption requirements
* Includes an adoption form

###  Volunteer

* Explains how users can help
* Includes a volunteer form

### ℹ About

* Information about the website
* Contact details


##  How to Run Locally

1. Clone the repository:

```bash
git clone https://github.com/SMushy14/smushy.github.io.git
```

2. Install dependencies:

```bash
bundle install
```

3. Run the site:

```bash
bundle exec jekyll serve
```

4. Open in browser:

```
http://localhost:4000
```
